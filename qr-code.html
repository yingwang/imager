Frequently Asked Questions

What is error correction?

QR codes include redundant data that allows them to be read even if part of the code is damaged or obscured. Higher error correction levels make the QR code more resilient but also larger. Low (7%), Medium (15%), Quartile (25%), and High (30%) indicate the percentage of the code that can be damaged while still being readable.

What's the maximum content length?

QR codes can encode up to approximately 4,296 alphanumeric characters or 2,953 bytes of data, depending on the error correction level. For best results and fast scanning, keep content under 300 characters.
